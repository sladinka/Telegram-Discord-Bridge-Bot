# Telegram-Discord bridge bot

## Функционал

- Автоматическая пересылка сообщений, с их файлами прикрепленными к сообщению и фарматированием, из вашего телеграмм канала в дискорд канал на вашем сервере.

## Установка и запуск

1. **Клонируйте репозиторий или скачайте файлы**

3. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Настройте переменные окружения:**
   - Скопируйте файл `env.example` в `.env`
   - Откройте `.env` и заполните следующие переменные:
     ```
     TG_TOKEN=токен_телеграмм_бота
     TG_CHANNEL_ID=токен_канала_в_телеграмме
     DC_TOKEN=токен_дискорд_бота
     DC_CHANNEL_ID=токен_канала_в_дискорде
     ```

4. **Добавление ботов**
Пригласите Discord бота на ваш сервер.
Добавьте Telegram бота в ваш канал и сделайте его администратором.

5. Запуск
```bash
python main.py
```
